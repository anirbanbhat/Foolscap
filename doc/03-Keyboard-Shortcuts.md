# Keyboard Shortcuts

## Files & windows

| Action | Shortcut |
|---|---|
| New document | ⌘N |
| Open file… | ⌘O |
| Open folder as workspace… | ⇧⌘O |
| Save | ⌘S |
| Save As… | ⇧⌘S |
| Close tab / window | ⌘W |
| Cycle tabs (MRU) | ⌃⇥ |
| Cycle tabs (MRU, backward) | ⌃⇧⇥ |
| Quit | ⌘Q |

## Editing

| Action | Shortcut |
|---|---|
| Undo / Redo | ⌘Z / ⇧⌘Z |
| Cut / Copy / Paste | ⌘X / ⌘C / ⌘V |
| Select All | ⌘A |

|---|---|

## Lines

| Action | Shortcut |
|---|---|
| Duplicate Line | ⇧⌘D |
| Delete Line | ⇧⌘K |
| Move Line Up | ⌥↑ |
| Move Line Down | ⌥↓ |

The **Edit ▸ Lines** submenu also has Sort, Trim Trailing Whitespace,
Tabs → Spaces, and Spaces → Tabs (no default shortcuts).

## Conversions

All the Convert actions live under **Edit ▸ Convert** and act on the current
selection (or the whole document if nothing is selected):

- UPPERCASE / lowercase / Title Case / iNVERT cASE
- camelCase / PascalCase / snake_case / kebab-case
- Base64 Encode / Decode
- URL Encode / Decode
- HTML Encode / Decode
